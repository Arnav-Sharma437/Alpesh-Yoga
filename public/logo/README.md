# Alpesh Yoga Logo Folder

Place your studio logo image file inside this folder.

### File Naming Convention:
- File Name: **`alpesh-logo`**
- File Format: Any standard web format (e.g., `.svg`, `.png`, `.jpg`, `.webp`)

### Example placements:
- `public/logo/alpesh-logo.svg` (Recommended for vector sharp lines)
- `public/logo/alpesh-logo.png` (Transparent background raster)
- `public/logo/alpesh-logo.jpg` (Solid background)

Once uploaded, the logo can be referenced in code relative to the root URL, e.g., `/logo/alpesh-logo.svg`.
